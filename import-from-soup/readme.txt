== Description ==

This plugin for Wordpress automatically getting your articles from soup.io

== Installation ==

1. Unzip the file
2. Upload it in /wp-content/plugins/ of your Wordpress installation
3. Activate the Plugin
4. Add to cronjob page: http://{YOUR_SITE}/wp-content/plugins/import-from-soup/wp-soup.php
4. That's All
